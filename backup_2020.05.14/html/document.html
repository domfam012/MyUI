시작하기

1. 소개

MY UI는 부트스트랩을 기반으로 제작되었습니다.
모든 플랫폼, 브라우저 또는 장치에 대한 최신 대시 보드를 가장 빠를게 구축할 수 있도록 도와드립니다.

2. 다운로드

MY UI를 다운로드하여 컴파일 된 CSS 및 JavaScript, 소스 코드를 얻거나 npm, RubyGems등과 같은 즐겨사용하는 패키지 관리자에 포함하십시오.

소스파일

- MyUI

설치(installation)

1. 저장소 클론

$ git clone https://github.com/domfam012/MyUI.git my-project

2. 프로젝트 디렉터리로 이동

$ cd my-project

3. npm 의존 모듈 설치

$ npm install
용법(usage)
$ npm run dev

개발을 위한 빌드를 watch 상태로 제공합니다.

$ npm run dist

minify 된 프로덕션 빌드를 제공합니다.

- MyUI(React)

설치(installation)

1. 저장소 클론

$ git clone https://github.com/domfam012/MyUI-react.git my-react-project

2. 프로젝트 디렉터리로 이동

$ cd my-react-project

3. npm 의존 모듈 설치

$ npm install
용법(usage)
$ npm run dev

로컬 호스트 개발 서버 시작(localhost:80)

$ npm run dist

minify 된 프로덕션 빌드를 제공합니다.

- MyUI(Vue)

설치(installation)

1. 저장소 클론

$ git clone https://github.com/domfam012/MyUI-vue.git my-vue-project

2. 프로젝트 디렉터리로 이동

$ cd my-vue-project

3. npm 의존 모듈 설치

$ npm install
용법(usage)
$ npm run dev //로컬호스트 개발 서버 시작(localhost:3000)
$ npm run build
$ npm run start //프로덕션 빌드 서버 시작
$ npm run generate //정적 프로젝트 생성

패키지 관리자

패키지 설치 후, node_modules 폴더에서 myui-bootstrap / myui-react / myui-vuejs 폴더를 복사 -> npm install

- npm

$ npm install myui-bootstrap

- react-npm

$ npm install myui-react

- vue-npm

$ npm install myui-vuejs

4. 빌드도구

MYUI에 포함된 npm 스크립트를 사용하여 소스코드를 컴파일하고 테스트를 실행하는 방법을 배우세요

사용 방법

- 부트 스트랩은 빌드 시스템에 npm 스크립트를 사용합니다. package.json에는 코드 컴파일, 테스트 실행 등 프레임 워크 작업을 위한 편리한 메소드가 포함되어 있습니다.

- 빌드 시스템을 사용하고 문서를 로컬로 실행하려면 MyUI의 소스 파일 및 노드 사본이 필요합니다.

- 의존성을 관리하는 데 사용하는 Node.js를 다운로드하여 설치하십시오.

- root / bootstrap 디렉토리로 이동하여 npm install을 실행하여 package.json에 나열된 로컬 종속성을 설치하십시오.

- 완료되면 명령 줄에서 제공되는 다양한 명령을 실행할 수 있습니다.

npm 스크립트 이용

- package.json에는 다음과 같은 명령과 작업이 포함되어 있습니다.

스크립트 설명
npm run dev 개발을 위한 빌드를 watch 상태로 제공합니다.
npm run dist minify 된 프로덕션 빌드를 제공합니다.
