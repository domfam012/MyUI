1. 소개

2. 다운로드

MY UI를 다운로드하여 컴파일 된 CSS 및 JavaScript, 소스 코드를 얻거나 npm, RubyGems등과 같은 즐겨사용하는 패키지 관리자에 포함하십시오.

소스파일

- MyUI

설치(installation)

1. 저장소 클론

$ git clone https://github.com/domfam012/MyUI.git my-project

2. 프로젝트 디렉터리로 이동

$ cd my-project

3. npm 의존 모듈 설치

$ npm install
용법(usage)
$ npm run dev

개발을 위한 빌드를 watch 상태로 제공합니다.

$ npm run dist

minify 된 프로덕션 빌드를 제공합니다.

- MyUI(React)

설치(installation)

1. 저장소 클론

$ git clone https://github.com/domfam012/MyUI-react.git my-react-project

2. 프로젝트 디렉터리로 이동

$ cd my-react-project

3. npm 의존 모듈 설치

$ npm install
용법(usage)
$ npm run dev

로컬 호스트 개발 서버 시작(localhost:80)

$ npm run dist

minify 된 프로덕션 빌드를 제공합니다.

- MyUI(Vue)

설치(installation)

1. 저장소 클론

$ git clone https://github.com/domfam012/MyUI-vue.git my-vue-project

2. 프로젝트 디렉터리로 이동

$ cd my-vue-project

3. npm 의존 모듈 설치

$ npm install
용법(usage)
$ npm run dev

로컬호스트 개발 서버 시작(localhost:3000)

$ npm run build
$ npm run start

프로덕션 빌드 서버 시작

$ npm run generate

정적 프로젝트 생성

5. 버튼(Button)

다양한 크기, 스타일을 지원한다.

1. 기본 버튼

2. 아웃라인 버튼

3. 아이콘과 같이 사용되는 버튼

4. 링크

1. 기본버튼

기본버튼은 5가지 사이즈(Extra Samll, Small, Medium, Lage, ExtraLarge)로 구성되어 있다.

기본 Extra Samll, Small, Medium 버튼 사이즈는 가로 80px 이며, 텍스트가 길어질 경우 좌/우 마진 10px를 유지한다.

Primary 컬러는 주요 기능/결정 버튼으로 사용한다.

2. 아웃라인 버튼

5가지 사이즈(Extra Samll, Small, Medium, Lage, ExtraLarge)로 구성되어 있다.

기본 Extra Samll, Small, Medium 버튼 사이즈는 가로 80px 이며, 텍스트가 길어질 경우 좌/우 마진 10px를 유지한다.

3. 아이콘과 같이 사용되는 버튼

버튼을 강조하거나 기능을 쉽게 이해시키기 위해 아이콘을 텍스트와 같이 사용한다.

아이콘과 텍스트의 간격은 6px이며, 아이콘+텍스트는 버튼 넓이의 중앙정렬한다.

4. 링크

텍스트에 링크를 걸었을 때 사용한다.

마우스 오버시 텍스트 하단에 라인이 생겨 링크가 있음을 알려준다.

1. 인풋박스

인풋박스는 흰색 배경에 스트록으로 표현된다.

일반적인 상태에서는 플레이스홀더를 사용하여 어떤 내용을 입력하는지 힌트를 준다.

입력 시 플레이스홀더는 사라지며 인풋박스의 테두리는 select 상태임을 나타내기위해 색상이 진하게 바뀐다.

미리입려된 내용이나 텍스트를 수정할 수 없는 Readonly 상태가 있다. (그레이 배경컬러에 스트록없음)

입력이 잘돗되었을 경우 밸리데이션으로 잘못된 내용을 알 려준다.

2. 셀렉트박스

조건을 정해놓고 조건을 고를 때 사용한다.

3. 파일선택 인풋박스

첨부파일을 선택할 때 사용한다.

파일을 선택하면 파일명은 자동으로 입력 된다.

4. 검색 인풋박스

검색을 할 때 사용한다.

13. 네비게이션 바(Navigation Bars)

웹사이트의 가로넓이 1200px를 기준으로 한다.
로고는 좌측 30px을 띄우며, 메뉴는 로고를 기준으로 100px 띄운다.
메뉴간 간격은 60px이며 메뉴 포커스시 메뉴 하단에 바가 생긴다.
바는 텍스트 길이보다 10px 크다.
메뉴명의 길이, 메뉴의 개수에 따라 메뉴사이 간격이 변동 될 수 있다.
